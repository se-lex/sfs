# Förordning (2012:559) om inkomstindex för år 2013

Regeringen fastställer inkomstindexet enligt [58 kap. 10–12 §§ socialförsäkringsbalken](https://selex.se/eli/sfs/2010/110#kap58.10) till 154,84 för år 2013.
