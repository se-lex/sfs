# Förordning (2017:1071) om balanstal för år 2018

## 1 §

Balanstalet enligt [58 kap. 14–20 §§ socialförsäkringsbalken](https://selex.se/eli/sfs/2010/110#kap58.14) ska vara 1,0395 för år 2018.
