# Förordning (2017:1070) om inkomstindex för år 2018

## 1 §

Inkomstindexet enligt [58 kap. 10–12 §§ socialförsäkringsbalken](https://selex.se/eli/sfs/2010/110#kap58.10) ska vara 170,73 för år 2018.
