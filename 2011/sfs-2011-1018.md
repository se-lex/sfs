# Förordning (2011:1018) om balansindex för år 2012

Regeringen fastställer balansindexet enligt [58 kap. 22-24 §§ socialförsäkringsbalken](https://selex.se/eli/sfs/2010/110#kap58.22) till 140,45 för år 2012.
